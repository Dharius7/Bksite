# Backend API

This directory contains the backend API for Coral Credit Bank LTD.

## Structure

- `nodejs/` - Node.js/Express backend (JavaScript)
- `python/` - Python/Flask backend

## Choose One Backend

You can use either the Node.js or Python backend. Both provide the same functionality.

## Node.js Backend

Located in `backend/nodejs/`

### Setup
```bash
cd backend/nodejs
npm install
npm run dev
```

### Features
- Express.js REST API
- User authentication
- Account management
- Transaction processing

## Python Backend

Located in `backend/python/`

### Setup
```bash
cd backend/python
pip install -r requirements.txt
python app.py
```

### Features
- Flask REST API
- User authentication
- Account management
- Transaction processing
